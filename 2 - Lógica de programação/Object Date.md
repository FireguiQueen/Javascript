# Date
O Date nada mais é do que uma função construtora. 
E através dele, podemos trabalhar com datas com extrema facilidade.

## Entendendo os parâmetros
Os parâmetros deste objeto seguem esta ordem.:
<ol>
    <li>Year </li>
    <li>Month </li>
    <li>Day </li>
    <li>Hour </li>
    <li>Minutes </li>
    <li>Seconds </li>
    <li>Miliseconds </li>
</ol>
